# Superstore Sales Streamlit App

This Streamlit app analyzes the Superstore Sales dataset with 10 interactive pages:

1. Overview
2. Sales Analysis
3. Profit Analysis
4. Customer Segments
5. Product Categories
6. Regional Performance
7. Shipping Analysis
8. Discount Impact
9. Top Products
10. Data Download

Navigate using the sidebar to explore each section.

## Dataset
The app uses the Superstore Sales dataset (see `data/superstore_sales.csv`).

## Requirements
- streamlit
- pandas
- matplotlib
- seaborn

Install requirements:
```
pip install -r requirements.txt
```

## Run the app
```
streamlit run app.py
```
